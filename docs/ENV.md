# Environment Variables Reference

See Section 10.1 of PRD for detailed origins.

- `NEXT_PUBLIC_SUPABASE_URL`: Supabase project URL (Web + Mobile)
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`: Supabase anon key (Web + Mobile)
- `SUPABASE_SERVICE_ROLE_KEY`: Supabase service key (Web server only)
- `RAZORPAY_KEY_ID`, `RAZORPAY_KEY_SECRET`, `RAZORPAY_WEBHOOK_SECRET`: Razorpay keys
- `NEXT_PUBLIC_ONESIGNAL_APP_ID`, `ONESIGNAL_REST_API_KEY`: OneSignal Push Notifications
- `INTERAKT_API_KEY`: WhatsApp notifications via Interakt
- `RESEND_API_KEY`: Email sending via Resend
- `N8N_WEBHOOK_SECRET`: Secure webhook token for N8n integrations
- `REVALIDATION_SECRET`: Secure webhook token for Next ISR revalidation
- `EXPO_PUBLIC_SUPABASE_URL`, `EXPO_PUBLIC_SUPABASE_ANON_KEY`: Expo aliases for Supabase
